# Convert absolute direction back to qualitative tool labels

Reverses the transformation done by
[`tri_absolute_direction()`](https://mcguinlu.github.io/triangulate/reference/tri_absolute_direction.md),
converting absolute directions ("left"/"right") back to labels like
"Favours comparator".

## Usage

``` r
tri_absolute_direction_invert(dat)
```

## Arguments

- dat:

  Data in long format
